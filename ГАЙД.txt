Первым делом - вам потребуются телеграмм аккаунты, ибо есть вероятность что их будут блокировать. Настоятельно не рекомендуется заниматься всем описанным ниже со своего основного аккаунта!!!

Проект состоит из 4-х основных файлов:
1) config.ini
2) DirectInviter.py
3) MessageSender.py
4) ParseTelegramUsers.py

1. config - это конфигурация куда вам нужно вписать необходимые данные. Вам потребуются:
- api_id его можно поулчить на сайте my.telegram.org в разделе API development tools*
- api_hash его можно поулчить на сайте my.telegram.org в разделе API development tools*
- username ваш ник в телеграмме (указывать его нужно без знака @ в начале ника)

* От вас потребуют войти по номеру телефона, на котором уже зарегистрирован телеграмм аккаунт. После входа вам нужно найти вкладку API development tools, она будет в центре экрана.
Все что нужно там заполнить - Это "Name", "Short Name" и нажать на галочку рядом с надписью "Desktop", после чего подтвердить выполнение, получить нужные данные и вставить их в конфиг.

2. ParseTelegramUsers - файл, который позволит вам получить список всех участников беседы (!!! Важно, именно беседы, группового чата. Получить список пабликов - не выйдет/не целесообразно)
и запишет их в отдельный новый файл, с ним вам взаимодействовать никак не нужно. Так же важно, чтобы аккаунт через который вы входили на сайте my.telegram.org находился в нужной вам беседе.

Изначально вам нужно будет повторный раз зайти на аккаунт телеграмм, но только уже через терминал (описал ниже). После чего от вас потребуют ссылку на нужный вам чат, с которого вы будете собирать
данные о пользователях. Готово!

3. MessangeSender - альтернативный скрипт, работает как обычная рассылка/спам по пользователям с которых вы уже собрали данные ранее. Перед самой рассылкой, в терминале запросит у вас текст сообщения
которое будет отправляться.

4. DirectInviter - Этот файл является еще одним методом добавления людей в беседу, только на этот раз он не рассылает сообщения, а напрямую добавляет пользователей в указанную вами беседу,
которую у вас запросят в терминале.


Как всем этим пользоваться?
- Вам понадобится установить на компьютер Python. На ютубе есть тонны гайдов.
- Для удобства вам понадобится приложение Visual Studio code
- Первым делом после открытия VSC - вам нужно найти в самом верху приложения вкладку Terminal -> Наведитесь на нее и выберите опцию New Terminal. Терминал у вас откроется снизу.
  В терминал вам нужно будет вписать данные команды по очереди:
pip install telethon
pip install asyncio
  Больше это прописывать не нужно, оно прописывается всего лишь один раз.
- Далее с боку в панельке вы выбирайте нужный вам сценарий (ParseTelegramUsers, MessageSender, или DirectInviter)
- После выбора сценария, сверху, где вы нашли опцию "Terminal" найдите еще одну, она называется "Run". Как только вы ее нашли, наведитесь на нее,
  вам выпадет меню где нужно выбрать первый вариант "Run without" Debugging
- Все получилось, сценарий выполнен!

Так же важен порядок выполнения. Перво-наперво вам нужно сканировать/парсить пользователей, по этому если у вас нет файла который заканчивается на .json - начните с него.
